# Python Quiz Game

Final project developed for the Python course at REDI School.

This project is a quiz game built with Python and Tkinter.  
The user answers multiple-choice Python questions and receives a final score at the end of the quiz.

# Features
- Graphical User Interface (GUI)
- Object-Oriented architecture
- Random quiz questions
- Multiple-choice answers
- Final score display
- Play again functionality
- CSV file processing
- External libraries integration

# Technologies Used
- Python 3
- Tkinter
- Pandas
- Pillow (PIL)


# REDI Requirements Covered
This project covers the following topics:

## Object-Oriented Programming (OOP)
- Structured using classes, methods, and encapsulated states (`QuizGame`)

## File Processing
- Reading quiz questions from a CSV file using pandas

## Requests and External Libraries
- Using external libraries such as:
  - pandas
  - Pillow (PIL)

## Error Handling and Exceptions
- Handling missing assets or files and invalid operations using try/except
